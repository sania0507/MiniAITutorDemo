Mini AI Tutor – Python GUI
Description

A simple Python-based tutor for K-12 Science and Math. Users type a question, and the tutor returns an answer from a hardcoded dataset. Demonstrates Python programming, Tkinter GUI, and basic Q&A logic.

Features
Interactive GUI with input box, “Ask” button, and answer display
Hardcoded dataset of educational Q&A

Works entirely offline — no internet or API required( i tried using an openapi key but apparently it requires billing and credits)

Usage
1. Open PowerShell and navigate to the project folder:
   cd C:\path\to\PROJECT1

2. Run the tutor:
   py .\tkinter_qa_app.py

Languages & Tools: Python 3.x, Tkinter
